### Development
1. Install Hugo
    ```
    brew install hugo
    hugo version
    ```

    More info: https://gohugo.io/getting-started/quick-start/

2. Start the server locally:
    ```
    hugo server -D
    ```
    Access it on http://localhost:1313/
